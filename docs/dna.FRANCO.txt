Franco's flagship active project is Ant on Mars (Hormiga en Marte) — a browser-based 3D vehicle game now running Three.js r183 WebGPU/TSL with Rapier physics, procedural biomes, zone capture mechanics, and a day/night/weather system (~4,000–5,000 line HTML). Franco has also produced structured prompt/manifest documents (ANT_ON_MARS_MANIFIESTO.md, ANT_ON_MARS_PROMPT_SONNET.md, ANT_ON_MARS_CONTEXT.md) to guide focused Claude Code sessions.
